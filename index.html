Surgimento da Internet

A internet que conhecemos hoje é o resultado de décadas de pesquisa, desenvolvimento e colaboração global. Ela continua evoluindo, a todo momento.

O surgimento da internet foi um processo gradual que envolveu diversas inovações tecnológicas e a colaboração de cientistas, engenheiros e pesquisadores de várias partes do mundo.
Na década de 1960, ocorreu o primeiro passo significado na criação da internet, o desenvolvimento da ARPANET (Advanced Research Projects Agency Network), proveniente da ARPA (Agência de Projetos de Pesquisa Avançada), do Departamento de Defesa dos EUA. A ARPANET permitia a comunicação de computadores em diferentes locais.
Nos anos 1970, surgiram os primeiros protocolos de comunicação que formariam a base da internet moderna: o Protocolo de Controle de Transmissão (TCP) e o Protocolo de Internet (IP). Esses protocolos padronizaram a forma como os dados eram transmitidos pela rede, tornando a comunicação entre computadores mais eficiente e confiável.
Na década de 1990, um cientista da computação britânico que trabalhava no CERN chamado Tim Berners-Lee, inventou a World Wide Web (WWW). A WWW, introduzida em 1991, facilitou a navegação na internet através de hipertexto e navegadores web, tornando a internet acessível a um público muito mais amplo. Navegadores como o Mosaic (1993) e o Netscape Navigator (1994) se tornaram populares, ajudando a democratizar o acesso à informação.

Tópicos explicativos:

  • O primeiro sistema de comunicação foi o Telégrafo, relacionado ao código Morse;
  • Em 1958 foi traçado um cabo de um continente ponta a ponta, pelo oceano;
  • Na década 1950 houve a Guerra Fria, o computador ENIAC foi usado para cálculos balisticos;
  • Também surgiu a DARPA, usando o ENIAC, uma agência de projetos e pesquisas avançada;
  • Surgiu a Arpanet:
    • Sistema de comunicação em pacotes, enviados um a um e que permitia múltiplos destinos;
    • Estudos sobre nós. Pontos de intersecção de informações funcionando como “checkpoints”. É o vô dos roteadores;
    • Para fins de pesquisas e fins militares;
    • Servia para trocar informações e conectar intalações de pesquisas e militares com o Pentágono, em 1960;
  • A primeira conexão (e queda) foi em 29 de outubro de 1969 entre a Universidade da Califórnia e o Instituto de Pesquisa de Stanford;
    • A primeira palavra trafegada na internet foi “LO”, de LOGIN, porém só foi enviado o “LO” antes da conexão cair;
  • Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (www);
    • O primeiro navegador foi chamado de WorldWideWeb;
    • Em 06/08/1991 foi criada a primeira página web;
    "A web não está concluída, é apenas a ponta do iceberg. As novas mudanças irão balançar o mundo ainda mais."
    - Tim Berners-Lee

Como funciona o HTML

Chamado de HyperText Markup Language, ou simplesmente HTML, é uma Linguagem de Marcação de Texto para criar e estruturar páginas web.
É composto por vários elementos, conhecidos como tags, que indicam ao navegador como exibir o conteúdo.

HTML é uma Linguagem de Programação?

A resposta é NÃO!
O que difere HTML das linguagens de programação (Java, C, Python, etc) é por ela ser uma linguagem de texto, de marcações, usada para descrever e estruturas páginas. Enquanto as linguagens de programação são usadas para criar a lógica que controla o comportamento dos programas.

O que são Tags?

Tags são usadas para definir e estruturar o conteúdo de uma página web. Servem como marcadores que indicam ao navegador como exibir o conteúdo.
Cada tag HTML começa com um sinal de menor que (<) e termina com um sinal de maior que (>). A maioria das tags vem em pares: uma tag de abertura e uma tag de fechamento. A tag de fechamento é precedida por uma barra (/).

Exemplos de Tags HTML:

  1. Tags de texto:
    • h1 a h6: Tags de título, cabeçalhos, do maior (h1) para o menor (h6).
    • p: Definição de um parágrafo.
  2. Tags de lista:
    • ul: Define uma lista não ordenada.
    • ol: Define uma lista ordenada.
  3. Tags de link e imagem:
    • a: Define um link.
    • img: Define uma imagem.
